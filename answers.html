Что такое и зачем нужен HTML?

HTML - язык разметки. Отвечает за структуру и содержание страниц сайта.

Что такое и зачем нужен Git?

Это распределенная система контроля версий. Нужна для того, чтобы была возможность работать командой над одним проектом.

Что такое SEO?

Это совокупность работ с целью улучшения позиций сайта в результатах выдачи поисковых систем, чтобы увеличить посещаемость сайта, тем самым способствовать раскрутке и продвижению сайта.

В чем отличие Git от GitHub?

GitHub - сайт, на котором хранятся репозитории, созданные в Git. Git - программа, осуществляющая контроль версий.

Что такое тег? Что такое атрибут?

Тег - элемент языка HTML, который имеет свой смысл и назначение. Атрибут - свойство тега.

Чем отличается язык разметки от языка програмирования? HTML можно назвать языком программирования? Аргументируйте свою позицию.

HTML нельзя назвать языком программирования, т.к. его задача заключается в разметке страницы и отображении данных. А задача языков программирования состоит в обработке данных и произведении различных вычислений.

Зачем нужен атрибут __alt__ у изображений?

В атрибуте _alt_ указывается информация о том, что изображено на картинке. Чтобы в случае, если по какой либо причине картинка не загрузилась, было понятно, что на ней изображено. Так же эта информация может быть воспроизведена для людей с ограниченными возможностями.

Зачем нужен тег `label`? Как объеденить `label` и `input`?

Тег `label` нужен для создания связи между элементом `label` и элементом формы. Благодаря этой связи можно изменять значение элементов формы нажав на текст (подпись) или другой элемент `label`. Для того, чтобы объеденить `label` и `input`, необходимо в теге `input` указать атрибут `id`, а в теге `label`атрибут `for` с таким же названием, как `id`.

Как объеденить радиокнопки в группу?

Необходимо задать всем радиокнопкам группы атрибут `name` с одинаковым значением.

Какие условия нужно соблюсти, чтобы отправить файл на сервер?

В теге `input` необходимо указать атрибут type="file". Далее, чтобы это поле заработало, нужно форме задать атрибут enctype="multipart/form-data" и указать атрибуты method="POST" и action, в котором прописать адрес программы или документа, который обрабатывает данные формы.

Что такое HTTP? Какие существуют HTTP методы?

HTTP - распространенный сетевой протокол передачи данных. HTTP-методы: POST (создать), GET (прочитать), PUT (обновить), DELETE (удалить).

Зачем используется атрибут name?

Задает имя полю формы (для сервера и якорей). Является обязательным атрибутом.

Что такое семантика? Зачем нужна?

Семантика (предназначение) - смысл, информационное содержание языка или его отдельной единицы. С помощью семантических тегов улучшается структура веб-страницы, добавляется смысловое значение той информации, которая в них заключена. Код становится чистым и читабельным.

В каких случаях используется тег `article`?

Когда контент связан семантически и автономен, т.е. содержимое `article` независимо от остального контента страницы.

Зачем используются ARIA-роли и ARIA-атрибуты?

ARIA-роли определяют тип элемента и указывают его роль на странице (значение). Существуют ARIA-атрибуты состояния и ARIA-атрибуты свойств, которые предоставляют дополнительную информацию о переопределенных элементах.

Зачем используются теги `iframe`, `audio`, `video`?

Тег `iframe` (индивидуальный или плавающй фрейм) позволяет вставить на текущую страницу сайта другую страницу или код, часто используется для добавления видео. Тег `audio` добавляет, воспроизводит и управляет настройками аудиозаписи на странице. Тег `video` добавляет, воспроизводит и управляет настройками видеоролика на странице.

За что отвечает свойство `display`?

Это свойство позволяет изменять тип элемента.

Чем отличается `width: 100%` от `width: auto`?

При `width: auto` ширина дочернего элемента автоматически постраивается под ширину родителя, в не зависимости от внесенных изменений (добавление отступов). При `width: 100%` ширина дочернего элемента выйдет за пределы родителя, если будут вносится изменения.

Зачем нужно свойство `box-sizing`?

С помощью этого свойства можно поменять алгоритм расчета ширины элемента.

Чем отличаются свойства `justify-content` и `align-items`?

`justify-content` - выравнивает элементы вдоль главной оси. `align-items` - вдоль поперечной оси.

Как заставить флексовые элементы переноситься на новую строку?

Задать свойство flex-wrap: wrap.

Назови два способа выравнивания: 1) блочного элемента по центру; 2) строчного элемента по центру.

Чтобы отцентрировать блочный элемент по центру, необходимо ему задать ширину, а затем задать внешние отступы слева и справа со значением auto.

Чем отличается абсолютное позиционирование от относительного?

При абсолютном позиционировании элементы выпадают из потока документов, и на их место становятся соседние элементы. При относительном позиционировании элементы остаются в потоке документа.

Что такое трек сетки в CSS Grid?

Grid Track - грид-интервал - пространство между двумя смежными вертикальными или горизонтальными грид-линиями.

Для чего используются свойства `grid-area` и `grid-template-area`?

Свойства `grid-area` задает название элементу, чтобы ссылаться на него с помощью шаблона сетки, который создается с помощью свойства `grid-template-area`.

В чем отличие свойств `justify-content`, `justify-items` и `justify-self`?

`justify-content` - выравнивает сетку по оси строки относительно контейнера. `justify-items` - выравнивает содержимое сетки вдоль оси строки. `justify-self` - выравнивает содержимое элемента (ячейки) вдоль оси строки.

Что такое единица _fr_?

fr (fractional unit) - единица гибкости. Если указывать ширину в fr, то можно добавлять сколько необходимо элементов, и их ширина будет равномерно разделена по отношению к контейнеру.

Зачем используются и как работают grid-column/grid-row?

Указывают позиции элементов в грид-сетке. grid-column - сокращенная запись от grid-column-start (задает первую вертикальную грид-линию для элемента) и grid-column-end (адает последнюю вертик. линию для элемента). Аналогично, grid-row - сокращение от grid-row-start (первая гориз.линия элемента) и grid-row-end (последняя гориз.линия элемента).

Что такое Flexbox и CSS Grid?

Flexbox - набор свойств, с помощью которых строятся сетки, работая по двум осям: главной и поперечной. CSS Grid - так же набор свойств для построения сеток, но в двумерном пространстве, можно выравнивать содержимое в обоих измерениях.

Чем отличается Flexbox свойство `justify-content` от `align-items`?

Свойство `justify-content` задает положение элементов по главной оси, а свойство `align-items` - по поперечной оси.

Зачем используется Flexbox свойство `flex-basis`?

Это свойство определяет базовый размер элемента.

Чем отличается `flex-grow` от `flex-shrink`?

Свойство `flex-grow` определяет, насколько отдельный flex-блок может быть больше соседних элементов, а свойство `flex-shrink` - насколько этот блок может уменьшаться относительно соседних элементов.

Чем отличаются CSS Grid свойства `justify-items`, `justify-content` и `justify-self`?

Свойство `justify-items` выравнивает все содержимое сетки по оси строки. Свойство `justify-content` выравнивает сетку по оси строки по отношению к контейнеру. Свойство `justify-self` выравнивает содержимое выбранного элемента по оси строки.

Зачем используется свойство `grid-area`?

Это свойство задает название элементу, чтобы на него можно было ссылаться с помощью шаблона сетки, созданного с помощью свойства `grid-template-area`.

Сокращением от каких свойств является свойство `grid-gap`? Сколько значений они принимают?

Свойство `grid-gap` задает расстояние между колонками и столбцами. Оно является сокращением от свойства `grid-column-gap`(расстояние между колонками) и свойства `grid-row-gap`(расстояние между строками). Принимают одно значение.
