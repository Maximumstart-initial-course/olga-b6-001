Что такое и зачем нужен HTML?

HTML - язык разметки. Предназначен для разметки и наполнения страниц сайта.

Что такое и зачем нужен Git?

Это распределенная система контроля версий. Нужна для того, чтобы была возможность работать командой над одним проектом.

Что такое SEO?

Это совокупность работ с целью улучшения позиций сайта в результатах выдачи поисковых систем, чтобы увеличить посещаемость сайта, тем самым способствовать раскрутке и продвижению сайта.

В чем отличие Git от GitHub?

GitHub - сайт, на котором хранятся репозитории, созданные в Git. Git - программа, осуществляющая контроль версий.

Что такое тег? Что такое атрибут?

Тег - элемент языка HTML, который имеет свой смысл и назначение. Атрибут - свойство тега.

Чем отличается язык разметки от языка програмирования? HTML можно назвать языком программирования? Аргументируйте свою позицию.

Зачем нужен атрибут __alt__ у изображений?

В атрибуте _alt_ указывается информация о том, что изображено на картинке. Чтобы в случае, если по какой либо причине картинка не загрузилась, было понятно, что на ней изображено. Так же эта информация может быть воспроизведена для людей с ограниченными возможностями.
